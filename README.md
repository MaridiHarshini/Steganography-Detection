# Steganography-Detection
A Python-based steganography detection system developed to identify hidden information inside image files using image processing techniques.

## Technologies Used
- Python
- OpenCV
- Tkinter
- SQLite

## Features
- Detect hidden messages in images
- User-friendly GUI interface
- Image upload and analysis
- Secure data handling
- Database integration for user management


## Duration
2025 – 2026

## How to Run
1. Install required libraries
2. Run `main.py`
3. Upload an image for analysis

## Applications
- Cybersecurity
- Secure communication
- Digital forensics
